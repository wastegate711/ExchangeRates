Программа позволяет получать курсы валют на заданную дату с сайта ЦБ или из базы данных.

В качестве СУБД используется MsSql. В папке DB находится скрипт для создания БД с пустыми таблицами и функцией. 
Таблица Currency предназначена для хранения списка валют. Таблица DailyRate предназначена для хранения курсов валют на каждую дату. 
Таблицы связаны отношением один-ко-многим. Функция GetRate принимает два аргумента - идентификатор валюты и дату, и возвращает курс. 
Для запуска проекта в файле App.config необходимо изменить строку подключения к СУБД 
source=user\SQLEXPRESS;

При первом запуске программы необходимо заполнить БД данными. Для этого предназначена кнопка "Заполнить БД".
При нажатии на кнопку таблица Currency заполняется списком валют, а таблица DailyRate курсами с 01.01.2021 по текущую дату.

На событие onLoad формы привязан обработчик события, который запускает таймер с интервалом 1 раз в час. 
Таймер проверяет, если текущий час=0 (т.е. наступил новый день), запрашивается с сайта ЦБ курс на новый день и записывается в БД.

При нажатии на кнопку "Запрос курса с сайта" загружаются данные с сайта ЦБ с помощью метода, принимающего два агрумента 
"дата" и "валюта", и возвращающего курс валюты.

Кнопка "Запрос курса из БД" производит запрос курса выбраной валюты из базы данных с помощью
хранимой функции. В качестве ORM используется EntityFramework.
